# Houston::TodoLists

Allows you to sync to-do lists with Todoist.


## Installation

In your `Gemfile`, add:

    gem "houston-todolists"

And in `config/main.rb`, add:

```ruby
use :todolists do
  # TODO: specify configuration options for Houston::TodoLists here
end
```

And then execute:

    $ bundle
